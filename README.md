# IoT-Based IV Fluid Monitoring System

## Overview

The IoT-Based IV Fluid Monitoring System is a smart healthcare solution designed to monitor intravenous (IV) fluid levels in real time. The system measures the remaining IV fluid, sends the data to the cloud, and provides timely alerts to healthcare staff when the fluid level falls below a predefined threshold, reducing the need for continuous manual monitoring.

---

## Objectives

* Monitor IV fluid levels continuously.
* Reduce the workload of healthcare professionals.
* Send real-time notifications when IV fluid is low.
* Enable remote monitoring through cloud connectivity.

---

## Features

* Real-time IV fluid monitoring
* Cloud-based data synchronization
* Low fluid level alerts
* Wireless IoT communication
* User-friendly monitoring dashboard
* Portable enclosure designed in SolidWorks
